# Brand Compliance & Editorial Review

You are performing a comprehensive editorial review to ensure content meets brand standards, quality requirements, and editorial excellence before publication.

## Review Process

Follow the 7-step editing process from `/agents/editor.md`:

### 1. First Read (Big Picture)
- Read content straight through without editing
- Assess overall effectiveness
- Note major issues
- Evaluate audience appropriateness

### 2. Structural Edit
- [ ] Organization: Logical flow and structure
- [ ] Completeness: All necessary sections present
- [ ] Balance: Appropriate emphasis on key points
- [ ] Transitions: Smooth flow between sections
- [ ] Length: Meets target word count

### 3. Content Edit
- [ ] Clarity: Clear and understandable
- [ ] Accuracy: Facts and claims verified
- [ ] Relevance: On-topic and valuable
- [ ] Examples: Concrete and appropriate
- [ ] Evidence: Claims properly supported

### 4. Brand Compliance Edit
Check against brand guidelines:
- [ ] **Voice**: Matches `/brand/voice-and-tone.md`
- [ ] **Tone**: Appropriate for content type and audience
- [ ] **Terminology**: Correct product names (check `/brand/terminology.md`)
- [ ] **Messages**: Aligns with positioning (check `/brand/messaging-framework.md`)
- [ ] **Legal**: No prohibited claims

### 5. Copy Edit
- [ ] Grammar, spelling, punctuation
- [ ] Syntax and sentence structure
- [ ] Style consistency

### 6. Format and Polish
- [ ] Headers: Proper hierarchy
- [ ] Lists: Parallel structure
- [ ] Links: Functional and descriptive
- [ ] Spacing and readability

### 7. Final Check
- [ ] Readability score (Hemingway grade 8-10)
- [ ] SEO checklist (if applicable)
- [ ] All required elements present
- [ ] Mobile-friendly

## Content-Specific Checklists

### For Blogs
- [ ] Hook in first 2 sentences
- [ ] Scannable with short paragraphs
- [ ] SEO optimized (keywords, meta)
- [ ] CTA present and clear
- [ ] 600-1,000 words

### For Customer Stories
- [ ] Compelling headline with customer + result
- [ ] Challenge-solution-results structure
- [ ] 3+ quantifiable metrics
- [ ] 2-4 authentic customer quotes
- [ ] 800-1,200 words

### For Social Posts
- [ ] Platform-appropriate length
- [ ] Clear CTAs
- [ ] Hashtags and emojis used strategically
- [ ] Tagged accounts correct

## Start Here

Ask the user:
1. What type of content needs review? (blog, customer story, whitepaper, social, etc.)
2. Where is the content located? (file path)
3. Is this a quick review or comprehensive edit?
4. Any specific concerns or areas to focus on?

## Output Format

Provide feedback as:
1. **Overall Assessment** - Ready to publish? Needs revision?
2. **Critical Issues** - Must be fixed before publication
3. **Important Improvements** - Should be addressed
4. **Suggestions** - Optional enhancements
5. **Brand Compliance Score** - Pass/Fail with details

Be specific: Reference line numbers, provide exact corrections, explain why changes are needed.
