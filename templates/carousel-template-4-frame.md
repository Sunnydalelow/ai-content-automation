# LinkedIn Carousel Template - 4-Frame Format

**Campaign**: [Campaign Name]
**Due Date**: [Date]
**Platform**: LinkedIn
**Dimensions**: 1080x1080px per frame (square)
**Total Frames**: 4
**Customer**: [Customer Name]

---

## Frame 1: Headline

**Layout**: Bold headline with key metric or hook

**Content**:
```
[KEY STAT OR METRIC]

[Supporting headline text]
[Customer name + product]
```

**Visual**:
- Large stat or number (hero element)
- Customer logo + your product logo
- Background: Brand gradient or colors
- Professional, impactful design

---

## Frame 2: Before

**Layout**: Problem state / Old way

**Content**:
```
BEFORE

x [Problem 1]
x [Problem 2]
x [Problem 3]
```

**Visual**:
- Red/orange warning tones
- X marks or prohibition icons
- Dark or muted background

---

## Frame 3: After

**Layout**: Solution state / New way

**Content**:
```
AFTER

[checkmark] [Solution/Benefit 1]
[checkmark] [Solution/Benefit 2]
[checkmark] [Solution/Benefit 3]
```

**Visual**:
- Green/blue positive tones
- Checkmarks
- Light, optimistic background

---

## Frame 4: Quote (or Tag)

**Content**:
```
"[Customer quote that captures
the transformation or key insight.]"

-- [Name]
   [Title, Company]
```

**Visual**:
- Quote as focal point
- Professional headshot or company logo
- Clean typography
- Your product logo at bottom

---

## Export Instructions

1. Create 4 square frames (1080x1080px each)
2. Use consistent brand colors throughout
3. Maintain visual hierarchy: Hook --> Transformation --> Proof
4. Keep text readable on mobile (minimum 24pt font)
5. High contrast for accessibility
6. Upload to LinkedIn as carousel post

---

## Design Principles

1. **Consistency**: Same fonts, colors, layout across all frames
2. **Hierarchy**: Each frame has one clear focal point
3. **Readability**: Test at mobile size
4. **Story arc**: Headline --> Problem --> Solution --> Proof
5. **Simplicity**: Less is more
