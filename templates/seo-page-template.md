### Content Optimization
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords distributed naturally
- [ ] Keyword density 1-2% (not stuffed)
- [ ] Content length 1500-2500 words
- [ ] Headers (H2, H3) include keywords
- [ ] Content answers search intent completely

### Technical SEO
- [ ] Title tag optimized (55-60 characters)
- [ ] Meta description compelling (150-160 characters)
- [ ] URL slug clean and keyword-rich

- [ ] Internal links to 3-5 related pages

- [ ] Mobile-responsive design
- [ ] Fast page load speed (<3 seconds)
